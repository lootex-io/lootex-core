## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

# install pnpm first
Windows: iwr https://get.pnpm.io/install.ps1 -useb | iex
Mac: brew install pnpm

# install Node 

# install typescript

```bash
$ pnpm install
```
Put .env into ./configs folder

## Running the app

```bash
# watch mode
$ pnpm run start:dev


```
